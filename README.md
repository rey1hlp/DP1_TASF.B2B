# Ejecutar

- Backend

Desde la carpeta de backend:
.\mvnw -DskipTests clean spring-boot:run "-Dspring-boot.run.main-class=com.tasf_b2b.planificador.PlanificadorApplication"

en macos
./mvnw -DskipTests clean spring-boot:run "-Dspring-boot.run.main-class=com.tasf_b2b.planificador.PlanificadorApplication"

- Frontend

Desde la carpeta de frontend:
npm install (la primera vez)
npm run dev
